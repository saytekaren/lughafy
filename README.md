Kenya's all-in-one AI translator for 68+ local languages. Offline speech-to-speech, camera scans, Sheng support. Built by **saytekaren** with Grok.

## Quick Start
1. Install Flutter: [flutter.dev](https://flutter.dev)
2. `flutter pub get`
3. `flutter run` (connect your phone)

## Features (v0.2)
- 10 languages live (Swahili, Kikuyu, Luo, etc.)
- Speech recognition & TTS
- Sheng cleanup
- Coming: Full 68 langs, camera OCR
